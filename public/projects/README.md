## Tips
- Recommended size: 1200×900px (4:3 ratio matches the card layout)
- JPEG at 85% quality keeps file size small
- GIFs work too — great for Soniq Space to show the audio reactivity
- On Mac: cmd+shift+4 for a region screenshot, or cmd+shift+5 for full screen
